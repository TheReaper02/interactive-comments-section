# Frontend Mentor - Interactive Comments Section

Short solution note for this project.

## Live Site

- Live URL: https://thereaper02.github.io/interactive-comments-section/
- Repo URL: https://github.com/TheReaper02/interactive-comments-section

## What I Built

- Rendered top-level comments and nested replies from local data
- Added comment and reply creation
- Added edit and delete flows (with confirmation modal)
- Added upvote/downvote behavior with vote constraints
- Added responsive layout and visual polish based on the style guide

## Tech Used

- React (Vite)
- Tailwind CSS
- Context + useReducer for state management

## Notes / Learnings

This project helped me practice complex UI state updates in React, especially for nested data (replies), conditional actions by current user, and keeping interaction logic predictable while improving responsive styling.
